# Weekly Report Web Application

## Overview
This project is a web-based application for managing weekly reports, user accounts, and administrative tasks. It includes PHP scripts, HTML templates, and static assets.

## Folder Structure
- `weeklyreport/`: Main PHP application files
- `HTML/templates/`: CSS and design assets
- `static/`: Static files (add as needed)
- `Downloads/`: Miscellaneous scripts, SQL files, and database backups

## Features
- User account management
- Report creation, editing, and deletion
- Admin evaluation and monthly summaries
- Export functionality
- CSRF protection

## Setup
1. Place the project files in your web server directory (e.g., `htdocs` for XAMPP).
2. Import the SQL files in `Downloads/` to set up the database.
3. Configure database access in `weeklyreport/db.php`.
4. Access the application via your browser (e.g., `http://localhost/weeklyreport`).

## Requirements
- PHP 7.x or higher
- MySQL/MariaDB
- Web server (Apache recommended)

## Usage
- Admins can manage reports and user accounts.
- Users can submit and edit their weekly reports.
